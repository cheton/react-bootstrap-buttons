### Buttons

```jsx
<Button>Default</Button>
<Button btnStyle="primary">Primary</Button>
<Button btnStyle="secondary">Secondary</Button>
<Button btnStyle="success">Success</Button>
<Button btnStyle="danger">Danger</Button>
<Button btnStyle="warning">Warning</Button>
<Button btnStyle="info">Info</Button>
<Button btnStyle="light">Light</Button>
<Button btnStyle="dark">Dark</Button>
<Button btnStyle="link">Link</Button>
```

### Sizes

```jsx

<Button lg btnStyle="secondary">Large button</Button>
```

```jsx

    <Button block lg btnStyle="primary">Block level button</Button>

```
